Arquivos
----------------------------------------------------------------------------------------------------
1. principal.py - cria os comparativos e salva no arquivo resultados
2. stemmersSamples.py - gera os arquivos com os conjuntos após aplicação dos stemmers
3. resultados - arquivo com os resultados do metodo de paice dados os stemms passados como parametro para os stemmers
4. words_sample.txt - arquivo com os conjuntos de palavras para a aplicacao do metodo de paice
5. stems_*.txt - arquivos contendo os conjuntos com a aplicacao dos stemmers que dao nome ao arquivo
